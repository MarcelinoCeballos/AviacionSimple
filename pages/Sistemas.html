SISTEMA NEUMÁTICO

El Sistema Neumático

Sistema que utiliza el aire como recurso para realizar trabajos

El Ala es la parte primordial de una máquina para ser considerada una aeronave. El Ala, es la parte de la aeronave destinada a proveer la fuerza de sustentación. El Ala, también se utiliza para el almacenaje de combustible y en determinados casos se sujetan a ella motores y trenes de aterrizaje. El aire fluye alrededor del ala y, en este proceso, se combinan tres leyes; Bernoulli, Newton y Coanda para darnos como resultado La Fuerza de Sustentación.

EL EMPENAJE

El Empenaje

La parte trasera, o cola, del avión

El Empenaje es la parte de la aeronave destinada a proveer estabilidad y, para ello, cuenta con dos superficies fijas, una vertical y otra horizontal, llamadas Estabilizador vertical y Estabilizador horizontal. Además, el empenaje cuenta con dos superficies móviles que permiten el movimientos de cabeceo y guiñada, llamadas Timón de profundidad y Timón de dirección. Existen diferentes tipos de configuración de empenaje pero todos cumplen las misma función.
